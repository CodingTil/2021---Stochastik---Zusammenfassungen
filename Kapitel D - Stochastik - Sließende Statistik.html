Problemstellungen der Schließenden Statistik

Ist T⁡:Rn→R\operatorname{T}: \R^n \rightarrow \RT:Rn→R eine Funktion der Stichprobe (X⁡1,…,X⁡n)(\operatorname{X}_1, \dots, \operatorname{X}_n)(X1​,…,Xn​), so wird T⁡(X⁡1,…,X⁡n)\operatorname{T}(\operatorname{X}_1, \dots, \operatorname{X}_n)T(X1​,…,Xn​) als Statistik bezeichnet.

  • Im Schätzkontext wird sie auch als Schätzer,
  • bei der Verwendung im Rahmen von Hypothesentests Teststatistik genannt.

Kann jedes Element P⁡∈P\operatorname{P} \in \mathcal{P}P∈P eindeutig durch die Angabe eines Parameter(-vektors) ϑ∈Θ⊆Rk\vartheta \in \Theta \subseteq \R^kϑ∈Θ⊆Rk identifiziert werden, so heißt das Verteilungsmodell parametrische Verteilungssannahme: P={P⁡ϑ∣ϑ∈Θ}\mathcal{P} = \{\operatorname{P}_\vartheta \mid \vartheta \in \Theta\}P={Pϑ​∣ϑ∈Θ}
Die Menge Θ\ThetaΘ der möglichen Parameter heißt Parameterraum.
Ist keine derartige Parametrisierung gegeben, wird das Modell als nichtparametrisches Verteilungsmodell bezeichnet.

Mittlerer quadratischer Fehler

Der mittlere quadratische Fehler MSE (mean square error) eines Schätzers ϑ^\hat{\vartheta}ϑ^ bzgl. des Parameters ϑ\varthetaϑ ist definiert durch MSE⁡ϑ(ϑ^)=E⁡ϑ(ϑ^−ϑ)2\operatorname{MSE}_\vartheta(\hat{\vartheta}) = \operatorname{E}_\vartheta (\hat{\vartheta} - \vartheta)^2MSEϑ​(ϑ^)=Eϑ​(ϑ^−ϑ)2

Sei ϑ^\hat{\vartheta}ϑ^ eine Schätzung für ϑ\varthetaϑ. Dann gilt: MSE⁡ϑ(ϑ^)=Var⁡ϑ(ϑ^)+(E⁡ϑϑ^−ϑ)2\operatorname{MSE}_\vartheta (\hat{\vartheta}) = \operatorname{Var}_\vartheta (\hat{\vartheta}) + (\operatorname{E}_\vartheta \hat{\vartheta} - \vartheta)^2MSEϑ​(ϑ^)=Varϑ​(ϑ^)+(Eϑ​ϑ^−ϑ)2

  • Der Term E⁡ϑϑ^−ϑ\operatorname{E}_\vartheta \hat{\vartheta} - \varthetaEϑ​ϑ^−ϑ heißt Verzerrung oder Bias des Schätzers.
